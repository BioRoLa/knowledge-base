New to BioRoLa? Start here:

1. Read the [Getting Started guide](docs/getting-started.md) for onboarding steps.
2. Review the [Software Setup](docs/software.md) to configure your development environment.
3. Check out the [Hardware Overview](docs/hardware.md) to understand our physical systems.
4. Explore [Simulation](docs/simulation.md) to run our robot in a virtual environment.
